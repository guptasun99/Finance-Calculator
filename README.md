# Finance Calculator Pro & Games

A modern web app with financial calculators (compound interest, loan, mortgage, investment, retirement, savings, ROI) and interactive finance games. Features dark/light mode, charts, SEO, and a dedicated games page for fun money learning. No signup—100% free and educational!

## Features
- Compound Interest, Simple Interest, CAGR, Loan EMI, Mortgage, Investment, Retirement, Savings, ROI calculators
- Interactive charts (Chart.js)
- Dark/Light mode toggle
- SEO optimized (meta tags, JSON-LD, social cards)
- Finance Games: Budget Blitz, Stock Market Simulator, Savings Race, Financial Trivia
- High score tracking (localStorage)
- Responsive and mobile-friendly

## File Structure
- `index.html` – Main calculator app
- `style.css` – App styling
- `script.js` – Calculator logic
- `games.html` – Finance games page
- `games.css` – Games styling
- `games.js` – Games logic
